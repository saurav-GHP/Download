About The Game

TPS Demo Game is a third-person shooter prototype developed using the Godot Engine. The game explores futuristic environments including advanced robotics facilities and deep-space technology stations.

Players control a character from a third-person perspective while navigating high-tech areas such as spaceship interiors, research labs, and cyber-security complexes.

The current version is an early trial build demonstrating core gameplay mechanics. Future versions will introduce improved graphics, enemy AI, weapons systems, larger maps, and missions.

This project was created by Team Saurav as a learning project to explore programming and game design.
